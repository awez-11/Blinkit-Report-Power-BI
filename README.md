# Blinkit Sales Analysis Dashboard

## Project Overview
This dashboard analyzes Blinkit's sales performance, outlet distribution, and item category trends using Power BI.

## Key KPIs
- Total Sales
- Average Sales
- Number of Items
- Average Rating

## Features
- Outlet Establishment Trends
- Item Type Analysis
- Outlet Size Distribution
- Outlet Location Analysis
- Fat Content Analysis

## Tools Used
- Power BI
- DAX
- Excel

## Key Insights
- Tier 3 outlets generated highest sales
- Fruits and snack foods were top-performing categories
- Medium outlet sizes contributed maximum revenue
